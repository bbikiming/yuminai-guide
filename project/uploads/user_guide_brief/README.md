# Yuminai 사용자 가이드 — 디자인 브리프 패키지

> **목적**: Anthropic의 Claude Design (또는 다른 디자인 도구/디자이너)에 그대로 넣어 **Yuminai 사용자 가이드 (튜토리얼/온보딩 문서/제품 사이트)**를 만들 수 있는 종합 자료 묶음.

## 사용 방법 (3가지)

### Option A — 클로드 디자인에 한꺼번에 던지기

1. 이 폴더 전체를 zip으로 묶거나 디렉토리 채로 클로드 디자인에 첨부
2. `08_DESIGN_PROMPTS.md`의 **프롬프트 #1 (전체 가이드 사이트)** 를 그대로 복사 붙여넣기
3. 디자인이 결과물 생성

### Option B — 챕터별 분리 작업

1. `07_USER_GUIDE_OUTLINE.md`에서 만들고 싶은 챕터 선택
2. 그 챕터에 해당하는 **개별 프롬프트** (`08_DESIGN_PROMPTS.md`의 #2~#8) 사용
3. 챕터별로 각각 디자인

### Option C — 디자이너 파트너에게 전달

1. 패키지 전체를 디자이너에게 전달
2. `01_PRODUCT_BRIEF.md`로 컨텍스트 잡고 시작
3. 일관된 시각 토큰은 `04_VISUAL_IDENTITY.md` 참조

## 가장 중요한 3개 파일 (시간 없을 때만 봐도 OK)

1. **`01_PRODUCT_BRIEF.md`** — 이 앱이 뭔지 한 페이지로
2. **`04_VISUAL_IDENTITY.md`** — 색상·폰트·아이콘 (앱과 통일)
3. **`08_DESIGN_PROMPTS.md`** — 바로 쓸 수 있는 프롬프트

## 빠른 사실 (디자이너가 가장 자주 묻는 것)

- **어떤 OS?** macOS 26 (Tahoe) 이상 — 데스크톱 전용
- **다크/라이트?** 둘 다 지원, 다크가 default. 가이드도 다크 우선 권장
- **언어?** UI는 한국어 100%
- **타깃 사용자?** 솔로 풀스택 개발자 1명 (본인 사용)
- **브랜드 색상?** 보라(#7C3AED) → 파랑(#3B82F6) 그라디언트 (앱 아이콘)
- **UI accent?** Bright cyan (#22C8E0) — AG2R 시안
- **Agent 색상?** Claude=오렌지(#D97706), Codex=틸그린(#10A37F)
- **앱 카테고리?** "바이브 코딩" 워크스페이스 (LLM 채팅 앱과 IDE 사이)
