# 06 — 키보드 단축키 카탈로그

> 가이드의 cheat sheet 페이지에 그대로 사용 가능. 그룹별 정리.

## 글로벌 (앱 전체)

| 키 | 동작 |
|---|---|
| `⌘N` | 새 워크스페이스 |
| `⌘W` | 현재 워크스페이스 닫기 |
| `⌘1` ~ `⌘9` | 워크스페이스 빠른 전환 (사이드바 1~9번) |
| `⌘P` | Command Palette / 워크스페이스·파일·노트·명령 검색 |
| `⌘,` | 설정 (Settings) |
| `⌘?` | 단축키 cheat sheet (Shortcut Help Sheet) |
| `⌘Q` | 앱 종료 (자식 프로세스 cleanup 후) |
| `⌘⇧E` | 워크스페이스 백업 내보내기 (export archive) |
| `⌘⇧I` | 워크스페이스 백업 가져오기 (import archive) |

## Workflow 단축키 (조합)

### "아침 시작"
1. `⌘1` (어제 마지막 워크스페이스로)
2. `⌘K` (새 세션 시작)
3. `@daily` + `⌘↵` (어제 daily note inject + 오늘 할 것 요청)

### "에이전트 핸드오프"
1. Claude 답변 받음
2. 메시지 아래 chip 클릭 (또는 통합 picker에서 Codex 선택)
3. 자동 prefix + agent 전환 → `⌘↵`

### "외출 전 셋업"
1. 큰 task 입력 + Composer footer에서 plan mode 선택
2. `⌘↵` 전송
3. 사용자 승인 후 자동 실행 시작
4. 외출 → Telegram 알림 자동 도착

### "PR 생성 + 머지"
1. `⌘⇧G` (Git commit sheet)
2. AI 메시지 생성 클릭
3. Commit
4. PR 생성 sheet
5. 머지

## 단축키 디자인 원칙

1. **Apple 표준 우선** — `⌘N`(new), `⌘W`(close), `⌘P`(palette/print), `⌘,`(settings) 등
2. **워크플로우별 그룹** — 같은 modifier가 같은 컨텍스트
3. **`⌘⌥` = 패널 토글** (default macOS 패턴)
4. **`⌘⇧` = sheet 열기** (commit/manager 등 전용 화면)
5. **단축키는 selectable text** — 가이드에서 단축키 표기는 코드 폰트 (`⌘N`)
