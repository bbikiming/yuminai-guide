채팅창에 “작업이 중간에 끊겼어요. 다시 시도할까요?” 메시지가 떠요. 자동으로 다시 시작하지는 않아요 — 어떤 상태에서 멈췄는지 직접 확인하시라고 일부러 그렇게 두었어요.

텔레그램

연결 표시등이 빨간색이에요

표시등을 누르면 오류 기록 창이 열려요. 인증 · 횟수 제한 · 네트워크 · 서버 · 메시지 형식 다섯 가지로 나누어 한국어로 보여드려요.

화면 배치

사이드바가 너무 좁아 보일 때

⌘⌥1로 사이드바를 잠시 숨기거나, ⌘⌥0으로 양쪽 패널을 한 번에 닫아 집중 모드로 들어가 보세요.

모델 선택

입력창에서 모델이 안 바뀌어요

최근 업데이트로 고쳐졌어요. 입력창 아래 [● 클로드 · Sonnet ⌄] 버튼을 누르면 에이전트와 모델이 한 번에 바뀝니다.

텔레그램 한도

텔레그램 응답 길이 한도를 넘었을 때

설정 → 텔레그램에서 한도 초과 시 동작을 고르실 수 있어요. 알림만 · 차단 · 짧은 답변으로 줄이기 중 선택할 수 있고, 마지막 옵션은 가장 짧은 응답 모드로 자동 전환돼요.

용어 사전

가이드에서 자주 나오는 말들을 쉬운 한국어로 풀어드려요.

작업 공간 Workspace
하나의 프로젝트 폴더에 대응하는 작업 단위예요. 사이드바에서 가장 먼저 보이는 칸이에요.
에이전트 Agent
유미나이 안에서 일하는 AI 도구예요. 지금은 클로드와 코덱스 두 명이 있어요.
에이전트 칸 Pane
한 작업 공간 안에 떠 있는 에이전트의 작업창이에요. 각자 대화 · 설정을 따로 가져요.
입력창 Composer
메시지를 적는 곳이에요. 왼쪽에 에이전트 색깔 띠가 있고, 아래에 모델 선택 버튼이 있어요.
검토 요청 Handoff
한 에이전트의 답을 다른 에이전트에게 “이거 한 번 봐줘”라고 넘기는 기능이에요. 답변 아래 버튼 한 번이면 돼요.
하네스 Harness
프로젝트 폴더에 살짝 끼워두는 작은 설정 묶음이에요. 규칙 · 에이전트 안내 · 명령어 · 자동 동작이 들어가요.
연결 표시등 Health Pill
사이드바 아래 텔레그램 연결 상태를 알리는 작은 알약 모양 표시예요. 초록 · 노랑 · 빨강으로 보여드려요.
도구함 Inspector
화면 오른쪽 패널이에요. 노트 · 명령어 · 연결 상태를 한 번에 볼 수 있어요. ⌘⌥2로 열어요.
계획 모드 Plan Mode
코드를 직접 실행하지 않고 계획만 세우는 모드예요. 텔레그램처럼 밖에서 보낸 요청에는 자동으로 이 모드가 켜져요.
자동 폴더 Smart Folder
조건에 맞는 작업 공간을 알아서 모아주는 폴더예요. “최근 7일에 작업한 것” 같은 식으로 채워져요.
스킬 Skills
자주 쓰는 지시문을 미리 만들어 둔 단축키예요. /test, /review처럼 한 단어로 부를 수 있어요.
실시간 답변 Streaming
에이전트 답이 한 글자씩 화면에 흘러나오는 모습이에요. 작은 점이 깜빡이며 진행을 알려드려요.
노트 보관함 Vault
옵시디언 노트가 모여 있는 폴더예요. 대화에 노트를 붙이거나 대화를 노트로 저장할 때 여기로 들어가요.
대화 기록 Conversation Log
작업 공간의 모든 대화가 시간 순으로 한 줄에 쌓이는 기록이에요. 작업 공간마다 따로 보관돼요.
